# Sistema de Vendas com PostgreSQL

## 📌 Descrição
Sistema simples de controle de vendas utilizando PostgreSQL, com procedure para automação de vendas.

## 🧠 Funcionalidades
- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Procedure com validações

## ⚙️ Como executar

1. Execute o arquivo ddl.sql
2. Execute o arquivo dml.sql
3. Execute o arquivo procedure.sql
4. Teste com:

```sql
CALL realizar_venda(1, 2);
```
